# 08 — Growth Loops

Superfans grows through compounding loops, not one-off campaigns. Every core action
(predict, post, follow, subscribe) is engineered to pull in or retain another user.
North Star: **Weekly Predicting Users (WPU)**.

---

## 1. Acquisition loops
- **Shareable prediction cards:** every prediction generates a branded, OG-rich card → shared to social → click → match landing → signup. (Primary viral loop.)
- **Match SEO pages:** every fixture has a public, fast, indexable Match Hub → organic search → conversion.
- **Creator import:** creators bring existing audiences → fans join to follow them.

## 2. Content loops
```
User posts analysis → others engage/comment → author gains followers/reputation
   → more reach for next post → attracts new readers → some sign up → post more
```
More content → more reasons to visit → more contributors.

## 3. Prediction loops
```
Match scheduled → user predicts → shares card → friends predict to compete
   → match resolves → reputation/leaderboard moves → return for next match
```
The habit core; drives WPU and daily return.

## 4. Creator loops
```
Creator monetizes → invests more → better content → more subscribers
   → platform earns 20% → reinvest in creator tools/distribution → more creators
```
Self-reinforcing supply of premium content (`07`).

## 5. Community loops
```
Community forms → members post/predict → activity attracts new members
   → critical mass → community becomes destination → invites more fans
```
Density compounds value (Reddit-style).

## 6. Referral loops
- Personal referral links + rewards (badges, Pro trial) for both sides.
- Competition framing ("beat your friends") makes inviting intrinsic, not bolted-on.

## 7. Retention loops
- Daily relevant hooks (your teams play, analyst posted, streak at risk, leaderboard moved) → return → predict → reputation update → reinforce.
- AI churn prediction (`06`) targets the highest-leverage nudge per user.

## 8. Virality model
- **K-factor levers:** % who share (shareable cards), invites sent (competition), conversion of recipients (fast match landing).
- Target each prediction/post to be a potential entry point. Measure K and CAC by loop.

## 9. SEO strategy
- Programmatic, indexable pages for **matches, teams, leagues, creators, communities** with realtime data, structured data (schema.org SportsEvent), and fan content → durable organic acquisition.
- Fast (edge-cached) + mobile-first = ranking + conversion.

## 10. Social strategy
- Native shareable artifacts (prediction cards, result reveals, leaderboard brags, accuracy milestones).
- Creator cross-posting; auto-generated highlight cards from AI previews/results.
- Presence where sports fans already are (X, TikTok, Reddit, YouTube) pointing back to Match Hubs.

## 11. Growth metrics
| Metric | Why |
| --- | --- |
| **WPU (North Star)** | Core engaged habit |
| DAU / WAU / MAU + stickiness (DAU/MAU) | Engagement depth |
| Activation rate (first prediction in session 1) | Onboarding health |
| K-factor / invites per user / share rate | Virality |
| Organic vs paid mix, CAC by channel | Acquisition efficiency |
| D1/D7/D30 retention | Habit strength |
| Free→Pro, Pro→Elite conversion | Monetization |
| Creator count / creator GMV | Supply & revenue |
| Community retention | Network density |

## 12. Growth dashboard
- **Top:** WPU trend, activation funnel, retention curves (D1/D7/D30 cohorts).
- **Acquisition:** channel mix, K-factor, share→signup conversion, SEO traffic.
- **Engagement:** predictions/day, posts/day, follows/day, match engagement.
- **Monetization:** subs, MRR, creator GMV, conversion rates, churn.
- **Network health:** follow-graph density, active communities, creator pipeline.
- Reviewed weekly; every initiative tied to a specific loop and a metric it should move.
